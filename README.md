# AI Productivity System | Executive Hub

This repository contains the **AI Productivity System | Executive Hub**, a custom-built dashboard for managing AI-driven productivity workflows, prompt engineering, and habit formation.

## Features
*   **Executive Dashboard**: Tracks mastery percentage and total time saved.
*   **Curriculum Management**: Tracks progress through modules and lessons.
*   **Prompt Sandbox**: An integrated environment for testing and versioning AI prompts.
*   **Weekly Audit Trigger**: A tool for resetting goals and maintaining long-term habit formation.
*   **Deep-Focus Mode**: A distraction-free UI toggle for intensive work sessions.

## Deployment
This system is designed to be hosted via GitHub Pages. It utilizes `localStorage` to save user data locally within the browser, ensuring privacy and persistence of your progress.

## Usage
1. Clone the repository.
2. Ensure you have the `index.html`, `style.css`, and `script.js` files in your root directory.
3. Push to your GitHub repository and enable "GitHub Pages" in your repository settings under the "Pages" tab.
